BEETL

BEETL is my version of a lightweight bug tracking system that helps you easily track the lifecycle of your issues/bugs from creation to resolution. The frontend was built using REACT.JS and backend with NODE&MONGODB. The backend REST API's were created using NodeJS/Express to perform the CRUD actions.

The functionality includes * Multiple user-role creation * Easy bug creation and management using Redux forms * Ability to quickly store/retrieve data from a MongoDB database*

At the end of this project I gained a lot of confidence to learn and implement a complex full-stack framework within a short turnaround time.

TOOLS: REACT, NODE/EXPRESS, MONGODB/MONGOOSE, JAVASCRIPT, MATERIAL-UI DESIGN

GREENEATS

GREENEATS is a mobile-responsive vegetarian restaurant/recipe finding guide. Being a vegetarian myself and seeing its benefits prompted me to create this app to inspire others.

The main features of this app are:

  • * Search for vegetarian recipes based on the ingredients you provide.
  • * Search for popular veg-friendly restaurants by city name.

The app pulls this data from public api's like Zomato and Edamam and presents the results to the user. During the development of this app, I got familiar with the handling of various public API's (both the advantages and the challenges). I also developed a better understanding of the data manipulation of the JSON responses I got back from the servers, so that I can create a neat front-end UI based on that information.

TOOLS: HTML5, JAVASCRIPT, JQUERY, BOOTSTRAP, AJAX, PHOTOSHOP

FRESHPLATE

Freshplate is a lightweight front-end framework that gives you the power to build fast and adaptable websites with contemporary styling. It was a capstone group project for my front-end web certification program.

My role involved creating pure Javascript functions and libraries for creating different components that Freshplate provides from the get-go. This framework helps you create a website withing minutes. Key features of this framework include:

  • * 12-column grids
  • * Provide classes for adding components like accordions, tabs, forms and tables
  • * Easy helper classes to create cool typography and backgrounds like parallax
  • * Responsive image addition with photo-gallery creation and slideshow options
  • * Adding an instagram feed to your page

Developing a whole framework from scratch taught me what the building blocks of a front-end library are. Also having previously worked in teams throughout my career helped me guide the rest of my teammates with code version-control, reviews . I also acted as project manager to manage everyone's daily tasks and conduct stand-ups.

TOOLS: HTML5, SASS, VANILLA JAVASCRIPT, PHOTOSHOP, GIT
