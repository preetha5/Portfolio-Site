Jobstar

SanDiego's responsive job search guide

This project was my first redesign student project and the reason I picked this was because the original website was way outdated. Also this site was a huge database that could potentially attract more users if the UI could be improved. The old site was bulky,outdated,erroneous and difficult to navigate. So this proved to be a nice challenge as capstone project.

Lot of UX principles was applied right from the beginning and most of the duration was spent on activities which included Needs Analysis, Card Sorting with the team, low/high fidelity prototyping, moodboarding and designing. After we had finally decided on the looks and color themes then the development proceeded.

Jobstar helped expand all the skills I had and get in-depth views for each area of development. The final site spoke for itself as it turned out to be a user-friendly and minimalistic contemporary website.
