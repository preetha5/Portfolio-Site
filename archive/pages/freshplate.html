FreshPlate

A lightweight/responsive web framework

FreshPlate is a lightweight HTML5/CSS3 framework that gives you the power to create web pages quickly with solid, contemporary styling. This project is a colloborative effort between four Web designers including myself. We are constantly improving and upgrading this framework to hopefully put it out in the marketplace someday. So stay tuned for upcoming updates!!

The FreshPlate framework is an efficient framework that lets you build dynamic website with very less code. Don't believe me ? Check out the demos here. It has a 12 column grid to customize your layouts the way you like. I had the opportunity to work on multiple javascript components in the framework which included galleries, slideshows and navigation methods.

Whether you need responsive navigation menus, carousels, icons or text overlays. You will find them all here along with instructions. Feel free to download and use the opensource framework and drop us a line using the contact page for feedback.
